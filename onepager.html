Friendship Bracelets

Origin

These bracelets can be traced back to China and Arabic Countries. They can be a symbol of long lasting love, companionship, and happiness. The colors that are used also add sentimental value.

Theory

There is a theory that friendship bracelets have a wish with them, and when they fall off, the wish comes true.

Directions

Cheveron: Cut Your String and Tie a Loop. Cut two 6-ft pieces of each color. ... Begin Knotting Your Bracelet. ... Continue Knotting the Bracelet With a Backward Knot. ... Continue Forwards and Backwards Knots. ... Repeat Steps Two Through Four to Continue Your Bracelet. ... Braid the Last Several Inches of Thread.

Candy Stripe: Cut each string to be about 36" long. Tie them together with an overhand knot. ... Take string A and make a forward knot on string B. Use string A to make a forward knot on string C. Then, use string A to make a forward knot on string D. ... Continue this pattern with each string in turn.

Spiral Staircase: Cut 3 different colored strings about 32 inches (81 cm) long and tie them in a knot about 2 inches (5 cm) from the ends.. Use string A and tie it in a knot around strings B and C. Continue this step until you want to switch colors, then Tie string B around strings A and C. Contine this rotation unti all three colors have been used.
